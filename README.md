# myriade-ussd-app-nodejs-generator

Myriade USSD App NodeJS Generator
=========

An npm Module to generate USSD apps for Myriade platform

## Installation

  npm install myriade-ussd-app-nodejs-generator --save

## Usage

  var generator = require('myriade-ussd-app-nodejs-generator')
      

## Tests

  npm test
